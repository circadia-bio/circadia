# circadia: Circadia Lab Colour Palettes and ggplot2 Theme

Provides the shared visual identity for the Circadia Lab R ecosystem:
colour palettes, ggplot2 theme, and ggplot2 scales. Palettes cover
qualitative, sequential, and diverging use cases anchored on the lab's
brand colours. Designed to be used alongside zeitR, slumbR, tallieR, and
syncR.
